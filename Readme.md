## Font Test

Destinado a testar criação de icones com fontes externas geradas manualmente.

<br/>

## Pré requisitos
- Node;
- npm (ou yarn);
- expo
- Expo GO (Opcional)

### Como executar o projeto?

  - Clone o repositório;
  - Instale todas as dependências com:

``` bash
    npm install 
```
ou
``` 
    yarn
```

- Execute o projeto com ```expo start```
